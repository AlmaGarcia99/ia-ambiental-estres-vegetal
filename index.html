Del Estrés Vegetal

Proyecto experimental de Inteligencia Artificial enfocado en analizar variables ambientales como humedad, temperatura y viento para detectar patrones de riesgo ecológico y construir sistemas preventivos basados en datos.

Ver Proyecto Abrir Notebook

Visión del Proyecto

Este proyecto nace desde una idea simple: una planta puede mostrar señales de estrés antes de deteriorarse. Utilizando Inteligencia Artificial, análisis ambiental y Machine Learning, el objetivo es detectar patrones críticos antes de que ocurra un problema mayor. La misma lógica puede evolucionar posteriormente hacia monitoreo forestal, agricultura inteligente y sistemas de prevención de incendios.

Variables Analizadas

💧

Humedad

Monitoreo de humedad del suelo para detectar condiciones de estrés ambiental.

🌡️

Temperatura

Análisis térmico ambiental para identificar escenarios de riesgo ecológico.

🌬️

Viento

Factor importante en propagación de sequedad y potenciales incendios.

☀️

Luz Solar

Evaluación de exposición solar y su impacto en condiciones ambientales.

Machine Learning

El sistema utiliza Random Forest Classifier para aprender patrones ambientales y clasificar niveles de riesgo. El modelo puede interpretar condiciones críticas y generar alertas preventivas.
